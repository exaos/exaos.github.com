摘要：牛津大学和伦敦帝国学院的科学家们设计了一种低成本、高灵敏的中子探测器，可用于识别核武器。中子探测器包含的发光部件称之为闪烁体，这是中子与之发生反应的地方。以前通常使用氦-3元素作为闪烁体。十多年前，美国的许多军工项目都能够生产氦-3，现在这些项目多数已经结束，并且氦-3在医用和科学研究上使用也越来越广泛，导致价格飞涨。该团队现在设计的探测器使用的是锂-6元素作为闪烁体，这种元素的自然界含量丰富，价格也很便宜。并且这种新的探测器可有效区分伽玛射线的信号，误报警的概率也比较低。

其实，这种中子探测器不算什么新鲜的东西。但如果在海关、远航等地方的危险器检测之类应用上得到广泛推广也不错。

[[http://dotsies.org/][Dotsies]] 是使用点代替常规字母书写的一种拉丁字母显示体系，每个字母由五个点的显隐来表示。
+ 想法一：制作坚排显示的字体。
+ 想法二：改进，添加定位标识，用于图形识别。

#+begin_html
<!-- link href='http://dotsies.org/dotsies.css' rel='stylesheet' type='text/css' -->
<script type=\"text/javascript\">
function dotsies_hover() {
  var l = document.createElement('link');
  document.getElementsByTagName('head')[0].appendChild(l);
  l.setAttribute('href', 'http://dotsies.org/dotsies.css');
  l.setAttribute('type', 'text/css');
  l.setAttribute('rel', 'stylesheet');
  var s = document.createElement('style');
  document.getElementsByTagName('head')[0].appendChild(s);
  s.innerHTML = ('body, p, div, span, a, li, textarea, pre, input {font-family: Dotsies !important}');
}
</script>
<a href=\"javascript:dotsies_hover()\">点这儿试试</a> Dotest 的样子：<a href=\"javascript:dotsies_hover()\">See it in action</a>.<br />

Do not close up you heart.
</pre>
#+end_html
