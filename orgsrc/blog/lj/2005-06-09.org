#+TITLE: 2005.06.05 MacEmacsJP 的 Emacs 中文输入问题与解决办法
#+AUTHOR: 心蛛
#+DATE: [2005-06-09 11:57:00]

2005.06.05 日 MacEmacsJP 所发布的 Carbon Emacs 在 Tiger 下会出现中文无法正确输入的问题，表现为使用 Cmd-Spc 呼叫出输入法后，输入的显示全为日文，输入的字符是乱码的日文片假。解决办法为：
1. 打开 Emacs 的菜单
   : Help --> Carbon Emacs Packages --> Open Site-Start Directory
2. 把文件 carbon-emacs-japanese-init.el 拖到 Emacs 的图标或者打开的窗口里，即可
   用 Emacs 打开文件。
3. 修改此文件如下：
   1) =(set-language-environment 'Japanese)= 中的 Japanese 改为 Chinese-GB
   2) 将所有的 sjis-mac 改为 chinese-iso-8bit
   3) 将 euc-jp-unix 改为 euc-cn-unix
   保存此文件后，重新启动 Emacs 即可。
