实验数据结果保存在文件 =a.json= 中，内容如下：
#+BEGIN_EXAMPLE
  { 
    "P1-PH": [[2, 216,  67], [262, 3832]], 
    "P2-PH": [[2, 179,  62], [196, 3839]], 
    "P3-PH": [[2, 192,  82], [203, 3837]], 
    "P4-PH": [[2, 222, 120], [243, 3842]] 
  }
#+END_EXAMPLE

在 Python 中这样读取文件
#+BEGIN_SRC python
  import json
  rdict = json.load( open("a.json","r") )
#+END_SRC

然后实验数据以 dict 形式保存在 rdict 中，其 key 为冒号前的字符串。
