#+TITLE: Emacs Carbon 越来越好用了
#+AUTHOR: 心蛛

用 Emacs Carbon 22.0.50.1 已经快一个月了，比原来的 21.3.50 感觉好多了。这个版本
是 MacEmacs JP 所支持的，参见: http://home.att.ne.jp/alpha/z123/emacs-mac-e.html

中文的 MacEmacs 用户有多少呢？至少周围的 Emacs 用户都非常之少，目前只发现一个非
常老的老师（七十了）在 Linux 和 Unix 下使用 Emacs （他是我见过的老师中少有的不会
用 Windows 的人，呵呵～～）。尽管反感日本人，但他们的确在软件方面做了许多工作是
我们国人应该感到惭愧的。我们一个中文文字标准都搞得那么乱， GB2312、GBK、GB18030、
ZH、Big5 等等，至今没有一款通用的免费字体 (zysong 18030 可能算是免费的字体了)，
大家用的都还是微软系统中的 SimSun 。大部分的人都还不知道 UTF 编码为何物，而我们
的支持 Unicode 编码的字库更缺乏。

希望我们的汉字能够统一，繁体和简体得到统一，编码得到统一，字库得到统一。对于计算
机而言，繁体字并不特别麻烦，反而熟悉繁体字，容易让我们认识古代的文字、继承古代的
一些文明的东西。当然，更希望 Unicode 得到更好的发展，全世界的文字都能够包括在内，
到时候就不必为各种编码和文字的事情费心了。

Emacs 原来的 mule 对多国语言的支持中汉字算是最差的。希望这种状况越来越好。

很喜欢 Emacs Carbon，尽管是日本人做的对 CJK 支持的优化等工作，对于优秀的东西，我
们应该好好学习。
