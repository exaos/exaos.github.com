常用的几个围棋站点：
1. PandaNet-IGS
   - 站点：igs.joyjoy.net
   - 端口：7777
   - 协议：IGS
   - 说明：IGS 即 Internet Go Server 的首字母缩写，始于欧洲，后转至美洲，后被卖
     给韩国，再被转卖给日本，即现在的 Panda-IGS。在世界上算是下棋的人最多的站点
     （指全球范围使用英文进行对奕的，不包括纯中文的站点）。
2. NNGS
   - 站点：nngs.cosmic.org
   - 端口：9696
   - 协议：NNGS
   - 说明：NNGS 是脱胎于 IGS 的基于 GPL 发布的一款围棋服务器软件，虽然它上面的用
     户没有 IGS 多，但基于 NNGS 的围棋服务器却非常多。这是 NNGS 的官方网站。
3. CWS 中国围棋网
   - 站点：cws.weiqi.net
   - 端口：9696
   - 协议：NNGS
   - 说明：中国围棋网的主对奕站点，也是软件 CWC（仅 Windows 下运行）默认的登陆站点。
4. CTN
   - 站点：weiqi.online.sh.cn
   - 端口：8888
   - 协议：
   - 说明：我连接不上这个站点。weiqi.online.sh.cn 这个域名会自动导向到"新民围棋
     网中国天元网"（weiqi.cngames.cn）。
5. Aurora
   - 站点：aurora.go.art.pl
   - 端口：9696
   - 协议：
   - 说明：无法连接。而 aurora.go.art.pl 是一个波兰语的站点。
6. EGF
   - 站点：server.european-go.org
   - 端口：6969
   - 协议：
   - 说明：无法连接。Croatia 的第一个围棋站点。
7. LGS 传奇网络围棋
   - 站点：lgs.taiwango.net
   - 端口：9696
   - 协议：NNGS
   - 说明：传奇网络围棋 lgs.taiwango.net 是用 big5 编码的台湾的站点。
8. WING 国际互联网围棋俱乐部
   - 站点：wing.gr.jp
   - 端口：1515
   - 协议：
   - 说明：World-wide InterNet Gokaisho 使用的是 WING Go Server 软件。
