* 浏览器或综合平台

** Mozilla SeaMonkey

- URL -- http://www.seamonkey-project.org/
- 简介：由 Mozilla 组织开发的集成式网络应用程序套件，包括浏览器、电子邮件客户
  端、新闻组及订阅客户端、IRC 聊天客户端和 HTML 编辑器。其前身即 mozilla 套件。
- 推荐插件： Enigmail

** Firefox

- URL -- http://www.firefox.com/
- 推荐插件： NoScript, AdBlock, FireGPG

** Chrome/Chromium

- Google Chrome -- http://www.google.com/chrome
- 开源项目 Chromium -- http://www.chromium.org/
  （注：被＊金＊盾＊封锁，无法正常访问。）
- Chromium OS -- http://dev.chromium.org/chromium-os \\
  注：这是基于 Chromium 开发的操作系统，主要用于手持式或手机设备中。

** Safari

- URL -- http://www.apple.com/safari/
- 简介：由 Apple Corp. 开发的浏览器，使用 WebKit 引擎。

** Opera

- URL -- http://www.opera.com

* 电子邮件客户端

** ThunderBird

- URL -- http://www.getthunderbird.com/
- 推荐 Enigmail 插件。

** mutt

- URL -- http://www.mutt.org/
- 简介：基于纯文本的邮件客户端，运行于 Unix/Linux 系统上，小巧高效。

** Emacs + gnus

- URL -- http://www.gnus.org/
- 简介：运行在 GNU Emacs 上的消息阅读器。

** Apple Mail

- 简单：这是 Mac OS X 系统中自带的邮件客户端。

* 即时消息客户端

** Pidgin

- URL -- http://www.pidgin.im/
- 简介：多协议的聊天客户端，支持 AIM, ICQ, Gtalk, Jabber/XMPP, MSN, Yahoo,
  Bonjour, IRC, QQ, Gadu-Gadu 等等。

** Adium

- URL -- http://adium.im/
- 简介：运行于 Mac OS X 上的开源即时消息客户端，它与 Pidgin 使用相同的 libpurple
  协议库。

** IPmsg

- URL -- http://www.ipmsg.org/
- 局域网内部进行即时消息传递的软件，同时可以在局域网内部相互之间传文件等操作。
- 衍生软件
  - XIP Messenger -- http://www.bsddiary.net/xipmsg/
  - g2ipmsg -- http://www.freshports.org/net/g2ipmsg/
  - 飞鸽传书 -- http://www.ipmsg.org.cn/
  - [[http://ishwt.net/en/software/ipmsg/][IP Messenger fo Mac OS X]]

* SSH 客户端及服务器端

** OpenSSH

- URL -- http://www.openssh.org/ or http://www.openssh.com/
- 简介：最流行的 SSH 客户端及服务器程序。

** Putty

- URL -- http://www.putty.org/
- 简介：由 Simon Tatham 最初为 Windows 平台开发的 SSH 及 Telnet 客户端程序。
- 相关软件
  1. Bitvise Tunnelier
  2. Bitvise WinSSHD -- Windows 下的 SSH, SFTP, SCP 服务器端程序。

** Xshell (Xmanager)

- URL -- http://www.netsarang.com/products/enterprise.html or
  http://www.xmanager.com/
- 简介：Xsession 是集成套件 Xmanager 中的一个程序。

* FTP/SFTP 客户端

** Xsftp (Xmanager)

参见 [[Xshell (Xmanager)]]

** WinSCP

- URL -- http://winscp.net
- 简介：Windows 环境下使用 SSH的开源图形化 SFTP 客户端。同时支持 SCP 协议。它的
  主要功能就是在本地与远程计算机间安全的复制文件。

** ThunderBird

参见 [[ThunderBird]]

* 远程桌面、VNC 或 X 客户端等

** rdesktop

- URL -- http://www.rdesktop.org/
- 简介：访问 Windows 下的远程桌面或 Terminal Service 的客户端程序，即 RDP 协议的
  客户端程序。可连接 Windows 2000 Server, Windows Server 2003, Windows Server
  2008, Windows XP/Vista, Windows NT Server 4.0

** grdesktop

- URL -- http://www.nongnu.org/grdesktop/
- 简介：是 rdesktop 的 GNOME 前端程序。

** xVNC

- URL -- http://xvnc.sourceforge.net/

** TightVNC

- URL -- http://www.tightvnc.com/
- 简介：使用 VNC 协议的远程桌面控制软件。
