* BNU bc: 任意精度的计算器

- PROPERTIES ::
  - URL     ::  http://directory.fsf.org/GNU/bc.html
  - License ::  GPL
  - Version ::  2001-03-18, 1.06
- 介绍 :: 'Bc' is an arbitrary precision numeric processing language. Its syntax
          is similar to C, but differs in many substantial areas. It supports
          interactive execution of statements. 'Bc' is a utility included in the
          POSIX P1003.2/D11 draft standard.
- 说明 :: This version does not use the historical method of having bc be a
          compiler for the dc calculator (the POSIX document doesn't specify how
          bc must be implemented). This version has a single executable that
          both compiles the language and runs the resulting 'byte code.'  The
          byte code is not the dc language.

  - Dev-Lang :: common lisp
- 介绍 :: 20 世纪 60 年代后期在 MIT 开发的计算机代数系统 Macsyma 是计算机代数计
