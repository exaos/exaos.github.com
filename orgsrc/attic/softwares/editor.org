* Emacs

- URL -- http://www.gnu.org/software/emacs/
- 简介 -- 经典软件，无非多言
- 相关网址
  - EmacsWiki -- http://emacswiki.org/
  - Emacs 中文网 -- http://emacs.cn/
  - dotemacs -- http://www.dotemacs.de/
  - Planet Emacsen -- http://planet.emacsen.org/
- 插件
  - Org-Mode -- http://orgmode.org/

* nano

- URL -- http://www.nano-editor.org/
- 简介 -- GNU nano 是 pico 文本编辑器的替代版本，而 pico 是 Pine email 组件中的
  一部分。它的目标是：尽可以提供 pico 所有的特性并包含增强的功能。
- 特点 -- 非常小巧，功能齐全。
- 缺点 -- 无图形界面等主流特色。

* UltraEdit

- URL -- http://www.ultraedit.com/
- 简介 -- 增强的文本编辑器，适用于程序员或网站开发人员。

* TextMate

- URL -- http://macromates.com/
- 简介 -- Mac OS 下的一款很不错的增强文本编辑器。

* Hexedit

- URL -- http://www.chez.com/prigaux/hexedit.html
- 简介：使用 16 进制格式查看并编辑文件，很小巧的工具。
