编译器套件、调试工具

GCC

  • URL – http://gcc.gnu.org/
  • GCC = the GNU Compiler Collection, 最流行的编程开发套，无须多言。

GDB

  • URL – http://www.gnu.org/software/gdb/
  • GDB: The GNU Project Debugger

Intel 编译套件

  • URL – http://software.intel.com/en-us/intel-compilers/

icc: Intel C++ Compiler

ifort: Intel Fortran Compiler

idb: Intel debugger

mkl: Intel Math Kernel Library

编程语言

C/C++

clang

  • URL – http://clang.llvm.org/
  • 简介：使用 LLVM 作为后端的 C/C++/Objective-C/Objective-C++ 编译器。

Fortran

g95

  • URL – http://www.g95.org/
  • 简介：Fortran 95/2003 编译器。

Python

Python

  • URL – http://www.python.org
  • Python 语言的官方版本。

ActivePython

  • URL – http://www.activestate.com/activepython/
  • 由 ActiveState 公司编译打包发行的 Python 软件包。

Portable Python

  • URL – http://www.portablepython.com/
  • 适用于 USB 存储设备上的移动式 Python 编程环境，针对 Windows 平台。

Shell

zsh

  • URL – http://www.zsh.org/
  • 集成了许多 bash, ksh 和 tcsh 中的特征的交互式 shell。

Pascal

Free Pascal

  • URL – http://www.freepascal.org/
  • Free Pascal (FPK) 是开源的 Pascal 编译器，支持 x86, PPC, ARM 及 Sparc 平台。

Functional Programming

SBCL – Steel Bank Common Lisp

  • URL – http://sbcl.sourceforge.net/
  • 简介：从 CMU CL 继承而来的 Common Lisp 开发套件。

Clozure CL

  • URL – http://www.clozure.com/clozurecl.html
  • 简介：继承 OpenMCL (原 Mac OS 上的 MCL) 而来的 Common Lisp 开发平台。

CMUCL

  • URL – http://www.cons.org/cmucl/

GHC (Haskell)

  • URL – http://www.haskell.org/
  • 简介：Haskell 旗舰编译器。

dmd

gdc

  • URL – http://www.dsource.org/projects/ldc
