TeX 相关软件

TeXLive

  • URL – http://tug.org/texlive/
  • 简介：一套综合的 TeX 出版系统，提供 Unix, GNU/Linux, Windows 多种平台上的可执行代码，包含了所有主要的自由的 TeX 相关程序、宏包、字体，并提供世界上多种语言的支持。
  • 备注
    • 针对 Mac OS X 平台，参见 MacTeX – http://www.tug.org/mactex/
    • 针对 Windows 平台，有一个国内整合的 CTeX 套件
      • http://www.ctex.org/CTeX
    • 针对 Windows 平台的专有套件 – http://www.miktex.org/

LyX

  • URL – http://www.lyx.org/
  • 简介：底层基于 TeX/LaTeX 的 WYSIWYM 文档处理器，开源免费。

TeXmacs

  • URL – http://www.texmacs.org/
  • 简介：自由的 WYSIWYW (What You See Is What You Want) 编辑平台，特别针对科学家设计。提供丰富的与其它程序关联的前端接口。
  • 备注
    • 版本 1.0.6 或 1.0.7 中文界面不正常 (TeXmacs] Font fireflysung 11pt at 300 dpi could not be loaded) 的解决办法
      1. 下载 opendesktop-fonts (ftp://ftp.opendesktop.org.tw/odp/ODOFonts/OpenFonts/)
      2. 将 odokai.ttf (以前名称为 fireflysung.ttf) 放置到 $HOME/.TeXmacs/fonts/truetype/ 下
      3. 将 odokai.ttf 改名为 fireflysung.ttf
      4. 重新启动 TeXmacs

JabRef

  • URL – http://jabref.sourceforge.net/
  • 简介：使用 Java 开发的 BiBTeX 前端程序，用于管理参考文献。

所见即所得类办公软件

MS Office

  • 由 Microsoft 开发的办公套件，不用多说。
  • 我使用过的版本
    1. Windows 平台 – Office 95/97/2000/XP/2003/2007,
    2. Mac OS X 平台 – Office 2004/2008
  • 使用很少，仅在需要与领导或者某些除 Office 之外就无法办公的人员交流时才用。

OpenOffice.org

  • URL – http://www.openoffice.org/
  • 简介：其前身是 StarOffice，被 Sun 公司收购以后，改名为 OpenOffice，并使用开源授权 LGPL 发布，目前是 Unix/Linux 界最流行的公办套件之一。OpenOffice.org 也是 OpenDocument 标准的制订者之一。

KOffice

  • URL – http://www.koffice.org/
  • 简介： KDE 下的集成办公套件。

iWorks

  • URL – http://www.apple.com/iwork/
  • 简介：由 Apple 开发的办公套件，运行于 Mac OS X 平台上，包含字处理 Pages、数据表处理 Numbers 和幻灯演示 Keynote。支持 Microsoft Office 文件格式。
