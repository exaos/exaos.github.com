随着互联网的普及，越来越多的服务等不需要在本地安装软件了。这些服务主要是基于 HTTP 协议的应用程序，简称网页程序。这些程序也可略分为如下这些种类：

  • 基于 HTML4 及以前版本的使用 JavaScript 的网页程序
  • 基于新的 HTML5+CSS3 的网页程序
  • 基于非网页标准需要第三方插件才能运行的网页程序，这第三方插件主要有
    1. Java Applet
    2. Adobe Flash
    3. ActiveX, 只能运行于 Windows 平台，安全性极糟糕却又偏偏应用比较普遍
    4. Microsoft SilverLight, 当然，仅限于 Windows 平台
    5. 其它

其实，简单一点地说，网页程序就是需要在流览器中才能运行的程序啦，无论是基于 HTTP 还是 Google 未来的 SPDY 协议也罢。

在线操作系统 – WebOS

根据 Wikipedia: Web Operating System 中的介绍，在线操作系统指基于 HTTP 协议的桌面应用，也称为 WebOS 或 Webtop。目前流行的 WebOS 通常采用 JavaScript 和 AJAX 技术开发，与目前炎热的云计算相结合，WebOS 有大的潜力。

Online Operating System

  • http://www.oos.cc/
  • Wikipedia: Online OS
  • 支持的浏览器： Mozilla Firefox 1.5 以上版本, Internet Explorer 7.0 以上版本。
  • 目前不支持 Safari

EyeOS

  • http://eyeos.org/
  • 简介：集成了办公套件、群件办公 (groupware) 应用程序和专门的网页应用开发技术工具包的 云计算平台 。使用 eyeOS 框架可以创建运行于网页桌面系统的应用程序。
  • 授权： AGPLv3

Glide OS

  • http://www.glidedigital.com/
  • 简介：综合的云计算解决方案。

amoebaOS

  • http://amoebaos.org/
  • 简介：免费的在线操作系统。以后添加试用报告。

WebX Online OS

  • http://www.andrewmin.com/webx/
  • 授权： GPLv3
  • 支持的浏览器：Firefox 3, IE 7 (部分), Konqueror (部分), Opera 9.5, Safari 3
