文件处理

常用打包、压缩与切割工具

7zip

  • http://www.7-zip.org/
  • High compression ratio in new 7z format with LZMA compression
  • 使用 LZMA 压缩法的新 7z 格式有非常高的压缩比率
  • 支持的压缩格式：
    • Packing / unpacking: 7z, ZIP, GZIP, BZIP2 and TAR
    • Unpacking only: ARJ, CAB, CHM, CPIO, DEB, DMG, HFS, ISO, LZH, LZMA, MSI, NSIS, RAR, RPM, UDF, WIM, XAR and Z.

tar

  • http://en.wikipedia.org/wiki/Tar_(file_format)
  • tar (取自 tape archive 的缩写) 是传统的 Unix 工具，原来是处理磁带数据的。

xz

  • http://tukaani.org/xz/
  • 简介：LZMA（Lempel-Ziv-Markov chain-Algorithm 的缩写）是2001 年以来得到发展的一个数据压缩算法。 XZ Utils 是使用 LZMA 算法的压缩工具，它取代了原来的 LZMA Utils, 目标是为 POSIX 系统提供一套标准的 LZMA 算法的压缩工具。

编码、格式转换等

iconv

  • iconv 是 GNU C 库的标准应用程序之一，功能是在不同的编码之间进行转换。

convmv

  • http://www.j3e.de/linux/convmv/
  • 在不同操作系统及文件系统之间进行交换文件时，通常会出现文件名编码混乱的情形，尤其在 Windows 和 Linux 之间。通常简体中文的 Windows 系统会使用 GB2312 (或 GBK, GB18030) 编码保存文件名，但这些文件名到 Linux 系统上则会显示为不可识别的编码，因为大部分的 Linux 系统会默认使用 UTF-8 作为文件名的编码。而使用 convmv 则可以方便地让文件名在各种编码之间进行转换。

fdupes

  • http://netdial.caribe.net/~adrian2/fdupes.html
  • 寻找或删除某个目录下的重复文件。有时候有些目录下会有成千上万个文件，会有许多名称不同但内容相同的文件，则可用此工具查找并删除重复的文件，从而达到节约空间的目的。

问题与解答

减少 VBox 虚拟硬盘尺寸的办法
  1. 在虚拟机时清除无用的文件，进行磁盘整理

  2. 在虚拟机中使用 SDelete.exe 程序将无用的磁盘空间清零 

                                            SDelete -c C:

    可以到这儿下载Sdelete。

  3. 关闭虚拟机，使用 VBoxManager 进行磁盘压缩 

                                            VBoxManager modifyvdi <File Name> compact

    VirtualBox 3.0 以后，上述命令变更为 

                                            VBoxManage modifyhd <Filename | UUID> --compact

Bochs

  • http://bochs.sourceforge.net/
  • 简介：使用 C++ 开发的高可移植的 IA-32 (x86) PC 模拟器，可运行许多流行的平台上。它包括 Intel x86 CPU 模拟器, I/O 设备及定制的 BIOS。

Xen

  • http://www.xen.org/
  • http://www.xen.org/files/Marketing/WhatisXen.pdf
  • The Xen® hypervisor, the powerful open source industry standard for virtualization, offers a powerful, efficient, and secure feature set for virtualization of x86, x86_64, IA64, ARM, and other CPU architectures. It supports a wide range of guest operating systems including Windows®, Linux®, Solaris®, and various versions of the BSD operating systems.

Qemu

  • http://www.qemu.org/ (被墙)
  • 处理器模拟器，与 Bochs, PearPC 近似，但速度更快且跨平台性良好。在 Linux 系统上使用内核加速器 kqemu 可使用 Qemu 的模拟能够接近真实电脑的速度。
