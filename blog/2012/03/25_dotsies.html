关于 dotsies 字体的想法

Dotsies 是使用点代替常规字母书写的一种拉丁字母显示体系，每个字母由五个点的显隐来表示。

  • 想法一：制作坚排显示的字体。
  • 想法二：改进，添加定位标识，用于图形识别。

<!– link href='http://dotsies.org/dotsies.css' rel='stylesheet' type='text/css' –> <script type="text/javascript"> function dotsies_hover() { var l = document.createElement('link'); document.getElementsByTagName('head')[0].appendChild(l); l.setAttribute('href', 'http://dotsies.org/dotsies.css'); l.setAttribute('type', 'text/css'); l.setAttribute('rel', 'stylesheet'); var s = document.createElement('style'); document.getElementsByTagName('head')[0].appendChild(s); s.innerHTML = ('body, p, div, span, a, li, textarea, pre, input {font-family: Dotsies !important}'); } </script> <a href="javascript:dotsies_hover()">点这儿试试</a> Dotest 的样子：<a href="javascript:dotsies_hover()">See it in action</a>.<br /> <pre> TRUST love even if it brings sorrow. Do not close up you heart. </pre>

2013-07-19: 如果把 Dotsies 字体中的点变成平板上的凸点，这种字体是否可以直接变成一种新盲文？那样，这种拼写体系就不仅适用于视觉，也同样适用于触觉了。
